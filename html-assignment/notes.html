5 html Tags Learned

HTML Anchors (< a >)
HTML anchors are used to link everthing together. Anchors are capable of linking to other websites, specific thing on your webpage, or andything on the web
Image Element (< img >)
This singular tag is used to link a picture onto your webpage.
Unordered List (< ul >)
This tag is used to make a list that has no particular order to it
Ordered List (< ol >)
This tag is used to put together a list that has a specific order to it.
Important Text Tag (< strong >)
This tag is used to bold a specific text to give it mor emphisis.
